OBJETIVO
--------
Agregar los datos reales de los participantes del proyecto
en la página de Participantes del sitio.


PASO 1 — Clonar el repositorio (si no lo tienes)
-------------------------------------------------
    git clone https://github.com/urielo96/voz_forense.git
    cd voz_forense


PASO 2 — Crear tu rama
-----------------------
    git checkout main
    git pull origin main
    git checkout -b feature/participantes


PASO 3 — Editar el archivo
--------------------------
Abre el archivo:
    extraccion/templates/participantes.html

Dentro encontrarás tarjetas de ejemplo con este formato:

    <div class="participante-card">
        <div class="participante-avatar">AB</div>
        <div class="participante-nombre">Nombre Apellido</div>
        <div class="participante-rol">Rol en el proyecto</div>
        <div class="participante-institucion">ENACIF — UNAM</div>
    </div>

Instrucciones:
  - Reemplaza cada tarjeta con los datos reales de cada participante.
  - En "participante-avatar" pon las iniciales (2 letras). Ej: "Ana Pérez" → AP
  - Duplica el bloque <div class="participante-card"> para cada persona.
  - Puedes cambiar el texto de las secciones ("Investigación", "Desarrollo")
    por los nombres reales de los grupos del proyecto.
  - Borra el bloque de comentarios <!-- INSTRUCCIONES --> cuando termines.
  - No toques ningún otro archivo.


PASO 4 — Guardar y subir
-------------------------
    git add extraccion/templates/participantes.html
    git commit -m "Agregar participantes del proyecto"
    git push origin feature/participantes


PASO 5 — Crear el Pull Request
-------------------------------
  1. Ve a https://github.com/urielo96/voz_forense
  2. GitHub mostrará un botón "Compare & pull request" — haz clic.
  3. Título del PR: "Agregar participantes del proyecto"
  4. En la descripción anota brevemente cuántas personas agregaste
     y en qué secciones las organizaste.
  5. Asigna el PR a Diego (urielo96) como revisor.
  6. Haz clic en "Create pull request".


NOTAS
-----
  - Puedes previsualizar el resultado corriendo el servidor local
    si tienes Python instalado:
        pip install -r requirements.txt
        python manage.py runserver
    Luego abre http://127.0.0.1:8000/participantes/

  - Si tienes dudas sobre los datos de algún participante,
    deja la tarjeta con el nombre y marca el rol como "Por confirmar".
