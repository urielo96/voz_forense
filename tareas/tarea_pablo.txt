======================================================

OBJETIVO
--------
Aplicar la identidad visual institucional al sitio:
logos de la UNAM y la ENACIF en el navbar, y paleta
de colores institucional en lugar de los colores genéricos actuales.


PASO 1 — Clonar el repositorio (si no lo tienes)
-------------------------------------------------
    git clone https://github.com/urielo96/voz_forense.git
    cd voz_forense


PASO 2 — Crear tu rama
-----------------------
    git checkout main
    git pull origin main
    git checkout -b feature/identidad-visual


PASO 3 — Agregar los logos
---------------------------
Crea la carpeta de imágenes:
    extraccion/static/img/

Coloca ahí los dos archivos (consíguelos con Diego):
    extraccion/static/img/logo_unam.png
    extraccion/static/img/logo_enacif.png

Recomendación: usa versiones en PNG con fondo transparente.
Tamaño sugerido: altura de 40-48px (el navbar mide 64px de alto).


PASO 4 — Actualizar el navbar
------------------------------
Abre el archivo:
    extraccion/templates/base.html

Encuentra esta sección (~línea 258):

    <a class="navbar-logo" href="{% url 'inicio' %}">
        <img src="/static/img/logo_enacif.png" alt="ENACIF" onload="this.classList.add('loaded')">
        ForVoz
    </a>

Reemplázala para mostrar ambos logos:

    <a class="navbar-logo" href="{% url 'inicio' %}">
        {% load static %}
        <img src="{% static 'img/logo_unam.png' %}" alt="UNAM">
        <img src="{% static 'img/logo_enacif.png' %}" alt="ENACIF">
        ForVoz
    </a>

Ajusta los estilos de .navbar-logo en el mismo archivo si los logos
quedan muy grandes o muy juntos (la propiedad es img { height: 36px; }).


PASO 5 — Cambiar los colores institucionales
--------------------------------------------
En el mismo archivo base.html, en el bloque <style>, busca y reemplaza
los colores genéricos por los colores de la UNAM:

    Color actual          →  Color institucional
    ─────────────────────────────────────────────
    #4a7c8a  (azul-verde) →  #003f8a  (azul UNAM)
    #3a6070  (hover)      →  #002d6b  (azul UNAM oscuro)
    #2a5060  (dark header)→  #002050  (azul UNAM más oscuro)
    #1a2a4e  (navy texto) →  #001a3e  (azul UNAM profundo)
    #eaf3f5  (fondo claro)→  #e8eef8  (azul claro UNAM)
    #f0f6f8  (fondo suave)→  #edf1fa  (azul muy claro UNAM)

Si el equipo decide usar el dorado UNAM (#b59a5a) para algún acento
(por ejemplo los headers de sección), también puedes aplicarlo.

Usa Ctrl+H (buscar y reemplazar) en tu editor para hacer los cambios
de forma rápida. Cada color aparece varias veces.


PASO 6 — Verificar visualmente
-------------------------------
Corre el servidor local para revisar que todo se vea bien:

    pip install -r requirements.txt
    python manage.py runserver

Abre http://127.0.0.1:8000 y navega por todas las páginas:
  - Inicio, Acerca de, Participantes, Publicaciones
  - Herramienta (paso 1, 2 y 3)

Verifica que los logos se vean bien en tamaño y que los colores
sean consistentes en todo el sitio.


PASO 7 — Guardar y subir
--------------------------
    git add extraccion/static/img/
    git add extraccion/templates/base.html
    git commit -m "Aplicar identidad visual institucional UNAM/ENACIF"
    git push origin feature/identidad-visual


PASO 8 — Crear el Pull Request
-------------------------------
  1. Ve a https://github.com/urielo96/voz_forense
  2. Haz clic en "Compare & pull request".
  3. Título del PR: "Identidad visual institucional UNAM/ENACIF"
  4. En la descripción incluye capturas de pantalla del antes y después
     (opcional pero muy útil para la revisión).
  5. Asigna el PR a Diego (urielo96) como revisor.
  6. Haz clic en "Create pull request".


NOTAS
-----
  - Solo debes modificar base.html y agregar imágenes.
    No toques ningún otro archivo Python ni de configuración.
  - Si el GitHub Actions intenta hacer deploy automático al hacer el PR,
    es normal — el deploy solo ocurre cuando se mergea a main,
    no al abrir el PR.
